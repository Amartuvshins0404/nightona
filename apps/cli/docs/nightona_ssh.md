## nightona ssh

SSH into a sandbox

### Synopsis

Establish an SSH connection to a running sandbox

```
nightona ssh [SANDBOX_ID] | [SANDBOX_NAME] [flags]
```

### Options

```
      --expires int   SSH access token expiration time in minutes (defaults to 24 hours) (default 1440)
```
