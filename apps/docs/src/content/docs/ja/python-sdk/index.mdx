Nightona の Python SDK は、Nightona サンドボックスとプログラム的にやり取りするための堅牢なインターフェースを提供します。

## インストール

pip で Nightona の Python SDK をインストールします:

```bash
pip install nightona
```

または poetry を使用します:

```bash
poetry add nightona
```

## はじめに

Nightona の Python SDK を使い始めるためのシンプルな例を示します。

<Tabs syncKey="language">
  <TabItem label="Sync" icon="seti:python">
    ```python
    from nightona import Nightona

    def main():
        # SDK を初期化（既定で環境変数を使用）
        nightona = Nightona()

        # サンドボックスを新規作成
        sandbox = nightona.create()

        # コマンドを実行
        response = sandbox.process.exec("echo 'Hello, World!'")
        print(response.result)

    if __name__ == "__main__":
        main()
    ```
  </TabItem>

  <TabItem label="Async" icon="seti:python">
    ```python
    import asyncio
    from nightona import AsyncNightona

    async def main():
        # SDK を初期化（既定で環境変数を使用）
        async with AsyncNightona() as nightona:
            # サンドボックスを新規作成
            sandbox = await nightona.create()

            # コマンドを実行
            response = await sandbox.process.exec("echo 'Hello, World!'")
            print(response.result)

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </TabItem>
</Tabs>

## 構成

SDK は環境変数を使うか、コンストラクタにオプションを渡して設定できます:

<Tabs syncKey="language">
  <TabItem label="Sync" icon="seti:python">
    ```python
    from nightona import Nightona, NightonaConfig

    # 環境変数を使用 (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
    nightona = Nightona()

    # 明示的な設定を使用
    config = NightonaConfig(
        api_key="your-api-key",
        api_url="https://app.daytona.io/api",
        target="us"
    )
    nightona = Nightona(config)
    ```
  </TabItem>

  <TabItem label="Async" icon="seti:python">
    ```python
    import asyncio
    from nightona import AsyncNightona, NightonaConfig

    async def main():
        try:
            # 環境変数を使用 (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
            nightona = AsyncNightona()
            # 非同期コードをここに記述
            pass
        finally:
            await nightona.close()

        # 明示的な設定を使用
        config = NightonaConfig(
            api_key="your-api-key",
            api_url="https://app.daytona.io/api",
            target="us"
        )
        async with AsyncNightona(config) as nightona:
            # ここにコードを記述
            pass

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </TabItem>
</Tabs>
