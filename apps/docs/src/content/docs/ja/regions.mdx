---
title: リージョンの選択
---

import { TabItem, Tabs } from '@astrojs/starlight/components'

Nightona は現在、以下のリージョンで利用可能です。

- 米国（US）
- 欧州（EU）

Nightona の[設定](/docs/ja/configuration)手順でリージョンを選択できます。

```python
from nightona import Nightona, NightonaConfig

config = NightonaConfig(
    target="us"
)

nightona = Nightona(config)
```

```typescript
import { Nightona } from '@nightona-co/sdk';

const nightona: Nightona = new Nightona({
    target: "eu"
});
```
