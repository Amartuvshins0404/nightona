## NightonaAuthenticationError

Error thrown when authentication fails (HTTP 401).

**Properties**:

- `errorCode?` _string_ - Machine-readable error code if available
    - _Inherited from_: `NightonaError.errorCode`
- `headers?` _AxiosHeaders_ - Response headers if available
    - _Inherited from_: `NightonaError.headers`
- `statusCode?` _number_ - HTTP status code if available
    
    - _Inherited from_: `NightonaError.statusCode`




**Example:**

```ts
try {
  for await (const sandbox of nightona.list()) {
    console.log(sandbox.id)
  }
} catch (error) {
  if (error instanceof NightonaAuthenticationError) {
    console.log(error.statusCode)
  }
}
```
