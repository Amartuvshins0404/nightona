## Config

Main class for a new Nightona::Config object.

### Constructors

#### new Config()

```ruby
def initialize(api_key:, jwt_token:, api_url:, organization_id:, target:, otel_enabled:, _experimental:)

```

Initializes a new Nightona::Config object.

**Parameters**:

- `api_key` _String, nil_ - Nightona API key. Defaults to ENV['NIGHTONA_API_KEY'].
- `jwt_token` _String, nil_ - Nightona JWT token. Defaults to ENV['NIGHTONA_JWT_TOKEN'].
- `api_url` _String, nil_ - Nightona API URL. Defaults to ENV['NIGHTONA_API_URL'] or Nightona::Config::API_URL.
- `organization_id` _String, nil_ - Nightona organization ID. Defaults to ENV['NIGHTONA_ORGANIZATION_ID'].
- `target` _String, nil_ - Nightona target. Defaults to ENV['NIGHTONA_TARGET'].
- `otel_enabled` _Boolean, nil_ - Enable OpenTelemetry tracing for SDK operations.
- `_experimental` _Hash, nil_ - Experimental configuration options.

**Returns**:

- `Config` - a new instance of Config

#### read_env()

```ruby
def read_env(name)

```

Reads a NIGHTONA_-prefixed environment variable using the same precedence
as the Config initializer: runtime ENV first, then .env.local, then .env.
Only names starting with NIGHTONA_ are accepted.

**Parameters**:

- `name` _String_ - The environment variable name. Must start with NIGHTONA_.

**Returns**:

- `String, nil` - The value of the environment variable, or nil if not set.

**Raises**:

- `ArgumentError` - If name does not start with NIGHTONA_.
