The Nightona Python SDK provides a robust interface for programmatically interacting with Nightona Sandboxes.

## Installation

Install the Nightona Python SDK using pip:

```bash
pip install nightona
```

Or using poetry:

```bash
poetry add nightona
```

## Getting Started

### Create a Sandbox

Create a Nightona Sandbox to run your code securely in an isolated environment. The following snippet is an example "Hello World" program that runs securely inside a Nightona Sandbox.

<Tabs>
  <TabItem label="Sync" icon="seti:python">
    ```python
    from nightona import Nightona

    def main():
        # Initialize the SDK (uses environment variables by default)
        nightona = Nightona()

        # Create a new sandbox
        sandbox = nightona.create()

        # Execute a command
        response = sandbox.process.exec("echo 'Hello, World!'")
        print(response.result)

    if __name__ == "__main__":
        main()
    ```
  </TabItem>

  <TabItem label="Async" icon="seti:python">
    ```python
    import asyncio
    from nightona import AsyncNightona

    async def main():
        # Initialize the SDK (uses environment variables by default)
        async with AsyncNightona() as nightona:
            # Create a new sandbox
            sandbox = await nightona.create()

            # Execute a command
            response = await sandbox.process.exec("echo 'Hello, World!'")
            print(response.result)

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </TabItem>
</Tabs>

## Configuration

The Nightona SDK can be configured using environment variables or by passing options to the constructor:

<Tabs>
  <TabItem label="Sync" icon="seti:python">
    ```python
    from nightona import Nightona, NightonaConfig

    # Using environment variables (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
    nightona = Nightona()

    # Using explicit configuration
    config = NightonaConfig(
        api_key="YOUR_API_KEY",
        api_url="https://app.daytona.io/api",
        target="us"
    )
    nightona = Nightona(config)
    ```
  </TabItem>

  <TabItem label="Async" icon="seti:python">
    ```python
    import asyncio
    from nightona import AsyncNightona, NightonaConfig

    async def main():
        try:
            # Using environment variables (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
            nightona = AsyncNightona()
            # Your async code here
            pass
        finally:
            await nightona.close()

        # Using explicit configuration
        config = NightonaConfig(
            api_key="YOUR_API_KEY",
            api_url="https://app.daytona.io/api",
            target="us"
        )
        async with AsyncNightona(config) as nightona:
            # Your code here
            pass

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </TabItem>
</Tabs>

For more information on configuring the Nightona SDK, see [configuration](/docs/en/configuration).
