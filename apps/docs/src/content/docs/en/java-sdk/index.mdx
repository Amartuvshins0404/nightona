The Nightona Java SDK provides a robust interface for programmatically interacting with Nightona Sandboxes. It targets Java 11+ and uses OkHttp and Jackson.

## Installation

### Gradle

Add the Nightona SDK dependency to your `build.gradle.kts`:

```kotlin
dependencies {
    implementation("io.nightona:sdk:x.y.z")
}
```

### Maven

Add the Nightona SDK dependency to your `pom.xml`:

```xml
<dependency>
  <groupId>io.nightona</groupId>
  <artifactId>sdk</artifactId>
  <version>x.y.z</version>
</dependency>
```

## Getting Started

### Create a Sandbox

Create a Nightona Sandbox to run your code securely in an isolated environment. The following snippet is an example "Hello World" program that runs securely inside a Nightona Sandbox.

```java
import io.nightona.sdk.Nightona;
import io.nightona.sdk.Sandbox;
import io.nightona.sdk.model.ExecuteResponse;

public class Main {
    public static void main(String[] args) {
        // Initialize the SDK (uses environment variables by default)
        try (Nightona nightona = new Nightona()) {
            // Create a new sandbox
            Sandbox sandbox = nightona.create();

            // Execute a command
            ExecuteResponse response = sandbox.getProcess().executeCommand("echo 'Hello, World!'");
            System.out.println(response.getResult());

            // Clean up
            sandbox.delete();
        }
    }
}
```

## Configuration

The Nightona SDK can be configured using environment variables or by passing a configuration object:

```java
// Using environment variables (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
Nightona nightona = new Nightona();
```

```java
// Using explicit configuration
NightonaConfig config = new NightonaConfig.Builder()
    .apiKey("YOUR_API_KEY")
    .apiUrl("YOUR_API_URL")
    .target("us")
    .build();
Nightona nightona = new Nightona(config);
```

For more information on configuring the Nightona SDK, see [configuration](/docs/en/configuration).
